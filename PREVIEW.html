🛠️ Cách sử dụng

Bước 1: Load extension

1. Mở Chrome → chrome://extensions/
2. Bật Developer Mode
3. Click "Load unpacked"
4. Chọn thư mục: /Users/duynk/Documents/project/Social/dist

Bước 2: Sử dụng

  • Click icon extension trên toolbar
  • Bật toggle cho Facebook hoặc Instagram
  • Vào facebook.com hoặc instagram.com
  • Thấy notification "🚀 [Platform] Toolkit Active" ở góc dưới bên phải

💻 Development

Build extension:

npm run build

Development mode (auto-rebuild):

npm run dev

Sau khi thay đổi code:

  • Build lại (nếu dùng npm run build)
  • Vào chrome://extensions/
  • Click reload icon ở extension
  • Test lại
